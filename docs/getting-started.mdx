# Getting Started

Welcome to **StatikAPI** — generate static JSON endpoints from simple files.

## Installation

```bash
pnpm add -D statikapi
```

Or use via the monorepo `packages/cli`:

```bash
node packages/cli/bin/statikapi.js build
```

## Quick Start

Create `src-api/index.js`:

```js
export default { hello: 'world' };
```

Build:

```bash
statikapi build --pretty
```

Outputs:

```
api-out/
└─ index.json
```

Preview:

```bash
statikapi preview --open
```
