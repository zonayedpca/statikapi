# Getting Started

Welcome to **StatikAPI** — generate **static JSON endpoints** from small JS/TS modules.

This guide shows two flows:

- **From scratch** using `create-statikapi`
- **Inside an existing repo** by adding the CLI and a `src-api/` folder

---

## 1) Scaffold a new project (recommended)

#### With pnpm
```bash
npx create-statikapi my-api
```

#### With yarn
```bash
npx create-statikapi my-api
```

#### With npm
```bash
npx create-statikapi my-api
```

You will be prompted for:
- **Template**: `basic`, `dynamic`, or `remote-data`
- **Package manager** and whether to install now
- Optional ESLint, TS, directories, and deploy presets

Then:

#### With pnpm
```bash
cd my-api && pnpm dev
```

#### With yarn
```bash
cd my-api && yarn dev
```

#### With npm
```bash
cd my-api && npm run dev
```

The dev server watches `src-api/` and serves a local **UI** at `/_ui` with a live manifest, JSON previews, and copy-ready code snippets.

---

## 2) Integrate into an existing project

1. Add the dependency (dev):
#### With pnpm
```bash
pnpm add -D statikapi
```

#### With yarn
```bash
yarn add -D statikapi
```

#### With npm
```bash
npm i -D statikapi
```

2. Create your **source directory**:
```bash
mkdir -p src-api && printf "export default { hello: 'world' }\n" > src-api/index.js
```

3. Add scripts to `package.json`:
```json
{
  "scripts": {
    "dev": "statikapi dev",
    "build": "statikapi build --pretty",
    "dev:headless": "statikapi dev --no-ui",
    "build:api": "statikapi build"
  }
}

```

4. Run it:
#### With pnpm
```bash
pnpm dev
```

#### With yarn
```bash
yarn dev
```

#### With npm
```bash
npm run dev
```

---

## Directory layout

```
src-api/            # your route modules
  index.js          # → /index.json
  users/[id].js     # → /users/1/index.json, etc (via paths())
  docs/[...slug].js # → /docs/a/b/index.json, etc (via paths())
api-out/            # generated JSON (gitignored)
```
