# Dynamic Routes

Use square brackets to mark a **single param**. Provide a `paths()` function to enumerate values.

```
src-api/users/[id].js  â†’ /users/1/index.json, /users/2/index.json, ...
```

## Example

```js
// src-api/users/[id].js
export async function paths() {
  return ["1", "2", "3"];
}

export async function data({ params }) {
  return {
    id: params.id,
    role: Number(params.id) % 2 === 0 ? "editor" : "viewer",
    generatedAt: new Date().toISOString()
  };
}

```

`paths()` returns an array of **strings** for the bracketed param.
