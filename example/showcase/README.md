# StatikAPI Showcase

Demonstrates:

- Static routes (JS/TS)
- Dynamic routes via `paths()` (JS/TS)
- Catch-all routes
- Default export value/function vs. `data(ctx)`
- Remote data fetch from JSONPlaceholder (build-time)
- TypeScript sources

## Run

From repo root:

```bash
pnpm -C example/showcase dev         # build & watch; opens preview UI at http://127.0.0.1:8788/_ui
pnpm -C example/showcase build
```
